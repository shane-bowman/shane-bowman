Xamarin Mobile App

Description: Developed a cross-platform mobile application for COSC295 using C# and Xamarin to track game matches. The app features a settings page for database reset with hardcoded games, an opponent list with context menu deletion and cascading match removal, a matches page with game and date pickers, and a games page displaying match totals, all integrated with a local database for dynamic data management.

Reflection: The Xamarin app honed my C# skills, focusing on UI and async operations. Debugging async match deletions tested my problem-solving, overcoming complex sync issues. Balancing deadlines enhanced my time management, prioritizing UI tasks. I plan to use Flutter for future apps, applying state management lessons.

Skills Demonstrated: C#, Xamarin, UI/UX design, database management, asynchronous programming, problem-solving, time management.
