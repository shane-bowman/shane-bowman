Xamarin Mobile App

Description: Developed a cross-platform mobile application for COSC295 using C# and Xamarin to track game matches. The app features a settings page for database reset with hardcoded games, an opponent list with context menu deletion and cascading match removal, a matches page with game and date pickers, and a games page displaying match totals, all integrated with a local database for dynamic data management.

Reflection: This project sharpened my C# and Xamarin skills, particularly in designing intuitive UI and managing asynchronous database operations. Debugging async calls for the opponent deletion feature, which required cascading match updates, was challenging but enhanced my problem-solving abilities. Balancing assignment deadlines with code quality improved my time management. I plan to explore Flutter for future cross-platform apps to leverage its robust framework, building on lessons learned in state management and error handling.

Skills Demonstrated: C#, Xamarin, UI/UX design, database management, asynchronous programming, problem-solving, time management.
