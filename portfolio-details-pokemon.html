Pokémon Web Application

Description: Developed a web application fetching Pokémon data from PokéAPI, featuring Google authentication, session-based recent activity tracking, and an image upload form with cookie-based form autofill. Users can view Pokémon lists, filter by type, and see detailed attributes.

Reflection: The Pokémon app boosted my JavaScript skills, working with PokéAPI integration. Resolving Passport authentication issues with a partner honed my communication, ensuring seamless collaboration. Adapting to new protocols under deadlines built my resilience. I plan to use WebSockets for real-time features, leveraging collaboration skills.

Skills Demonstrated: JavaScript, Node.js, Passport authentication, PokéAPI, session management, cookie handling, UI/UX design.
